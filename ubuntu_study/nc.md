# nc 命令

Linux nc 命令用于设置路由器。

执行本指令可设置路由器的相关参数。

## 语法

```sh
nc [-hlnruz][-g<网关...>][-G<指向器数目>][-i<延迟秒数>][-o<输出文件>][-p<通信端口>][-s<来源位址>][-v...][-w<超时秒数>][主机名称][通信端口...]
```

- -n 直接使用IP地址，而不通过域名服务器。
- -u 使用UDP传输协议。
- -v 显示指令执行过程。
- -w<超时秒数> 设置等待连线的时间。

## 示例

```sh
# 扫描指定端口
nc -nvv 192.168.1.2 80
# 扫描 udp 端口
nc -uv 192.168.1.2 80
```
